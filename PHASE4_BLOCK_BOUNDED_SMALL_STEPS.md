# Phase 2 UX Clarity Improvements

This build improves parent/student usability before Phase 3.

## Student View

- Simple daily home page.
- Visible date.
- Today’s Plan only.
- Progress Map moved off the home page.
- Practice Again uses visual dots instead of confusing attempt counts.
- Encouragement changes when the child answers incorrectly.

## Parent View

- Recommendation shows exact lesson, skill, and reason.
- If parent does nothing, the app uses the recommendation automatically.
- Parent may manually assign Quick Warm-Up by level and lesson range.
- Future levels are locked; current and lower levels are available.
- Parent Settings are compact and secondary.
- Reset and passcode tools do not occupy the main Parent Dashboard.

## Daily Work Record

- Date range filters.
- Level filter.
- Separate line graphs for progress and repetition/practice.
