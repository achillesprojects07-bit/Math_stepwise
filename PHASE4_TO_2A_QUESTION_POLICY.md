# Phase 2 Student Information Edit Fix

This update keeps Current Level and Current Lesson controlled by app progress while making parent-editable profile fields clearly editable.

## Parent-editable fields
- Student Name
- Date of Enrollment
- Starting Level, limited to available/current/lower levels
- Parent / Guardian
- Notes

## App-controlled fields
- Current Level
- Current Lesson

## UX fix
Student Information now opens in read-only summary mode with an obvious **Edit Student Info** button. Edit mode displays inputs and a **Save Student Info** button.


## Encouragement cleanup
Removed the extra child-facing line ‘Look again when you are ready.’ Wrong-answer feedback now uses only the varied encouragement message at the top, with visual response on the selected answer.
