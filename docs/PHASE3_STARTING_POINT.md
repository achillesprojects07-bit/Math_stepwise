# Phase 3 Starting Point Update

This build adds a proper starting point workflow.

## Rules added

- The app no longer assumes the student must always begin at 6A-1.
- On first use, the parent/placement setup chooses the student's starting point.
- For this Phase 3 build, Level 6A is the fully built curriculum, so the starting lesson can be chosen from 6A-1 to 6A-200.
- Higher levels are shown as future phases and remain disabled until their curriculum is built.
- Current Level and Current Lesson remain app-controlled after setup.
- Student Information allows editing of profile fields and the saved starting point, but not the current app-controlled placement.
- Parent Reset returns the child to the saved starting point, not automatically to 6A-1.

## Carry-forward rules

- Practice Again continues until all practice items are correct.
- Parent View remains protected by passcode.
- Parent Settings remains compact and secondary.
- Daily Work Record retains filters and separate line graphs.
- Child-facing screens avoid technical wording and avoid secondary look/ready prompts.
