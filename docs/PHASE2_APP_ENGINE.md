# Phase 2 — App Skeleton / Engine

Phase 2 builds the first working app shell. It does not complete all Level 6A curriculum content yet; that happens in Phase 3.

## Included in Phase 2

- Student View
- Parent View
- Level dropdown
- Level 6A lesson map preview
- Lesson runner
- Multiple choice answer flow
- Progress saved in browser localStorage
- Practice Again / Review queue
- Mastery engine file
- Question generator file
- Progress storage file
- Phase 1 rules/config carried forward
- Audit script

## Student-facing principle

The child-facing app stays simple, encouraging, and uncluttered. Mastery rules run behind the scenes.

## Parent-facing principle

The parent view shows progress, review items, and recent attempts without overwhelming the screen.

## Phase 3 will add

- Full Level 6A curriculum behavior
- Better generated question variety
- Tap-to-count mode
- Flashcard mode
- Expanded 6A lesson mapping for all 200 lessons
- Stronger correction flow
