# Phase 2 Final Workflow Notes

## Student Home

- Shows today's date.
- Shows Today's Plan.
- Uses a single Start Today's Work button.
- Keeps the lesson map off the home page.
- Progress Map is available as a separate view.

## Progress Map

- Color-coded tiles show mastered, review, current, and locked lessons.
- Future lessons are locked.
- The child cannot jump ahead using the map.
- Lesson labels use the clean format: 6A-1, 6A-13, 6A-200.

## Parent View

- Parent View is protected by a parent code.
- Demo parent code is 1234.
- Parent Reset Student Progress is inside Parent Settings only.
- Reset clears all Math Stepwise progress keys from localStorage.

## Practice Again

- Practice Again uses wrong-first-try and slow-correct items.
- Practice Again continues until every practice item is correct.
- Wrong practice attempts do not clear the item.
- The child sees simple, encouraging wording.

## Daily Work Record

- Date range filters are included.
- Level filter is included.
- Default graph view is entire progress.
- Progress Graph and Practice & Repetition Graph are separate.
