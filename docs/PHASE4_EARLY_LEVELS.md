# Phase 4 — Early Levels Expansion

This build adds working early levels: 5A, 4A, 3A, and 2A, while preserving the full Level 6A build.

## Included levels

- 6A: Counting, number reading, and dot recognition up to 10
- 5A: Number reading up to 30, sequences up to 30/40/50, large numbers
- 4A: Number tracing/writing practice, number typing, and numbers up to 50
- 3A: Numbers up to 100/120, adding 1, adding 2, adding 3, adding up to 3
- 2A: Review up to 3A, adding 4 through 10, adding up to 10

## Key system improvements retained

- Starting point can be chosen from built levels 6A, 5A, 4A, 3A, 2A
- Current Level and Current Lesson are app-controlled after setup
- Reset returns to the chosen starting point
- Future unbuilt levels remain locked
- Parent Quick Warm-Up can use current or lower/completed levels
- App recommendations proceed automatically if parent does nothing
- Practice Again continues until all items are correct
- Student-facing wording stays simple and calm
