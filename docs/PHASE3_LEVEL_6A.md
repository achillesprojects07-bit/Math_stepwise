# Phase 3 — Level 6A Full Curriculum Build

This build expands Level 6A into a complete 200-lesson digital curriculum structure.

## Level 6A sections

- 6A-1 to 6A-30: Counting up to 5
- 6A-31 to 6A-100: Counting up to 10
- 6A-101 to 6A-150: Number Reading up to 10
- 6A-151 to 6A-200: Number of Dots up to 10

## UX rules carried forward

- Student Home remains simple and shows today’s date.
- Progress Map is separate from Student Home.
- Visible lesson labels use 6A-1 format.
- Practice Again shows only the varied encouragement message and visual progress dots.
- The removed secondary prompt wording is not included.
- Parent View remains code-protected.
- Parent Settings remains compact and secondary.
