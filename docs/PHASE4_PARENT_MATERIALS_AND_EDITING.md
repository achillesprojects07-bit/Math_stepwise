# Phase 4 Addendum — Student Info Editing and Learning Materials

## Student Information

Student Information remains editable after it has already been saved. Parents can update student name, enrollment date, starting level, starting lesson, parent/guardian, and notes.

Current Level and Current Lesson remain app-controlled.

If the parent changes Starting Level or Starting Lesson, the app shows a confirmation warning because this re-places the student and resets current progress to the new starting point.

## Table of Learning Materials

Parent View includes a Table of Learning Materials section. Parents can choose a built level and see content blocks. Clicking a block shows worksheet-equivalent lessons in that range.

Built levels available in Phase 4: 6A, 5A, 4A, 3A, 2A.
