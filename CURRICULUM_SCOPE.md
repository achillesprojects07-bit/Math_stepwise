## Level Display Rule

Because there are many levels, the student view should use a level dropdown instead of showing every level as a large card.

The app should show one selected level at a time.

## First Build Scope

Phase 1 locks the rules only.

Phase 2 will build the reusable app skeleton/engine.

Phase 3 will build Level 6A MVP.

## Level 6A Scope

Level 6A has 200 lesson IDs:

- 6A001–6A030: Counting up to 5
- 6A031–6A100: Counting up to 10
- 6A101–6A150: Number reading up to 10
- 6A151–6A200: Number of dots up to 10

SCT is not used for 6A, so time is tracked but is not a hard mastery gate.

## Timing Categories

### No-SCT Foundational Levels

- ZI
- ZII
- 6A
- 5A

Timing is observed but not used as a strict mastery gate.

### SCT-Based Levels

- 4A onward, where the table provides SCT values

Timing is part of mastery.
